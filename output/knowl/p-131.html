In Activity 1.3.3 we considered the vector field \(\vF(x,y) = \langle y^2,2xy+3\rangle\) and two different oriented curves from \((1,0)\) to \((-1,0)\text{.}\) We found that the value of the line integral of \(\vF\) was the same along those two oriented curves.

  1. Verify that \(\vF(x,y) = \langle y^2,2xy+3\rangle\) is a gradient vector field by showing that \(\vF = \nabla f\) for the function \(f(x,y) = xy^2 + 3y\text{.}\)

  2. Calculate \(f(-1,0)-f(1,0)\text{.}\) How does this value compare to the value of the line integral \(\int_{C_1}\vF\cdot d\vr\) you found in Activity 1.3.3?

  3. Let \(C_3\) be the line segment from \((1,1)\) to \((3,4)\text{.}\) Calculate \(\int_{C_3}\vF\cdot d\vr\) as well as \(f(3,4)-f(1,1)\text{.}\) What do you notice? \saveCount
