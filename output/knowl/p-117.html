We first note that since \(\vF\) is orthogonal to \(C_2\text{,}\) \(\int_{C_2}\vF\cdot d\vr=0\text{;}\) threfore \(\int_C\vF\cdot d\vr = \int_{C_1}\vF\cdot d\vr\text{.}\) We can parameterize \(C_1\) as just \(x\vi+\vj\) for \(0\leq x\leq 3\text{,}\) which leads to

\begin{equation*} \int_{C_1}\vF\cdot d\vr = \int_0^3\langle x,0\rangle\cdot \langle 1,0\rangle\, dx = \int_0^3 x\, dx = \frac{9}{2}\text{.} \end{equation*}
